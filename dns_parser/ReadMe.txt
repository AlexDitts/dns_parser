Парсер предназначен для парсинга товаров со всеми характеристиками с сайта dns.ru.

Порядок установки:
У вас должен быть установлен язык python версии >= 3.9
создаём виртуальное окружение:
python -m venv parser_venv
Активируем окружение.

После скачивания, находясь в директории dns_parser в командной строке набираем (устанавливаем необходимые зависимости.
$ python -r requerements.txt


Способ применения:
В файле parser.py для функции multiprocess_get_list_links указываем аргументы (url категории товаров, которые собираемся
парсить, указываем количество страниц)
В функыии multiprocess_get_product вторым аргументом указываем название категории товаров. Это название будет указано во
вновь созданной директории, куда будут записываться файлв с изображением товаров, а так же в названии json файла
с данными о товарах. Первый аргумент функции не трогаем.
Запускаем файл parser.py. Получаем файл <category_name.json> и директорию с файлами изображения товаров.
Для наглядности данные также выводятся в консоль.
